Architecture

This system consists of 12 independent microservices:

  • customer-service (port 8001) - Customer management
  • staff-service (port 8004) - Staff management
  • manager-service (port 8005) - Manager management
  • catalog-service (port 8006) - Category management
  • book-service (port 8002) - Book CRUD
  • cart-service (port 8003) - Shopping cart
  • order-service (port 8007) - Order processing
  • pay-service (port 8008) - Payment processing
  • ship-service (port 8009) - Shipping management
  • comment-rate-service (port 8010) - Reviews & ratings
  • recommender-ai-service (port 8011) - Book recommendations
  • api-gateway (port 8000) - This web interface
